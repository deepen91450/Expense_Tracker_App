# 💰 Expense Tracker App (Android | Firebase)

A **Smart Expense Tracker Android Application** developed to help users manage daily expenses, track spending, and analyze financial habits using graphical insights.

This project was built as a **Third Year B.Sc Computer Science academic project** under the **University of Mumbai** and also aligns with real-world application development practices.

---

## 📌 Project Overview

The **Expense Tracker App** helps users record and monitor their daily, weekly, and monthly expenses efficiently.

Instead of manual tracking (diaries/Excel), this app provides a **digital solution with automated calculations and analytics**.

📖 According to the project documentation, the system:

* Eliminates manual expense tracking
* Maintains a digital expense diary
* Generates reports using graphical representation 

---

## 🚀 Key Features

* 👤 User Registration & Login (Firebase Authentication)
* 💸 Add / Update / Delete Expenses
* 📊 Daily, Weekly & Monthly Analytics
* 🥧 Pie Chart Visualization (Expense Distribution)
* 🗂️ Category-based Expense Tracking
* 📜 Transaction History
* 💰 Budget Management System
* ⚡ Real-time database updates (Firebase Realtime DB)

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/deepen91450/Expense_Tracker_App
```

### 2️⃣ Open in Android Studio

## 📈 How the App Works

1. User registers/login via Firebase
2. User adds expenses under categories
3. Data is stored in Firebase database
4. App calculates:

   * Daily expenses
   * Weekly expenses
   * Monthly expenses
5. Displays analytics using pie charts

---

## 📊 Database Design

The system includes:

* **User Table**
* **Expense Table**
* **Budget Table**
* **Category Table**

These store:

* User details
* Expense amount, date, category
* Budget allocation
